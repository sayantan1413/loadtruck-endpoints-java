#### Endpoints

1. POST "/load"
2. GET "/load" (Expects a shipperId)
3. Get "/load/{loadId}" (Expects a Id)
4. PUT "/load/{loadId}" (Expects a Id)
5. DELETE "/load/{loadId}" (Expects a Id)
